# Konga Invalid Card Test

Automated test for validating invalid card payment scenarios on Konga e-commerce platform using Selenium IDE.

## Purpose
This test project was created to ensure that Konga's e-commerce platform properly handles and validates invalid credit card information during the payment process. The goal is to verify that the system correctly rejects invalid card details and provides appropriate error messages to users, preventing fraudulent transactions and improving user experience.

## Description
This project tests how Konga's payment system handles invalid credit card information during the checkout process.

## Tools
- Mozilla Firefox Browser
- Selenium IDE extension installed
- Github

## How to Run the Test

1. Install Selenium IDE extension in your browser
2. Open Selenium IDE
3. Click "Open an existing project"
4. Select the .side file from this repository
5. Click the play button to run the test

## Test Steps
1. Navigate to Konga.com
2. Log in with test credentials
3. Add item to cart
4. Proceed to checkout
5. Enter invalid card details
6. Verify error message appears

## Expected Result
The system should reject the invalid card and display an appropriate error message.
